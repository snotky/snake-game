# 貪食蛇

## 啟動方式

### 方法一：直接用瀏覽器開啟（最簡單）

```bash
open index.html
```

或直接雙擊 `index.html` 檔案，用任何瀏覽器開啟即可。

---

### 方法二：用 Python 起本地伺服器

```bash
# Python 3
python3 -m http.server 8080
```

開啟瀏覽器前往：`http://localhost:8080`

---

### 方法三：用 Node.js (npx serve)

```bash
npx serve .
```

開啟瀏覽器前往提示的網址（通常是 `http://localhost:3000`）。

---

## 操作說明

| 按鍵 | 功能 |
|------|------|
| ↑ / W | 向上 |
| ↓ / S | 向下 |
| ← / A | 向左 |
| → / D | 向右 |
| P | 暫停 / 繼續 |
| R | 重新開始 |

## 遊戲規則

- 吃到**紅色食物** → +10 分，蛇變長
- 吃到**金色獎勵食物** → +30 分（每長 5 節出現一次，限時！）
- 撞牆或咬到自己 → 遊戲結束
- 最高分會自動儲存在瀏覽器中

## 難度選擇

開始前可選擇速度：**慢 / 中 / 快 / 極速**
