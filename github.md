# GitHub Pages 部署指南

## 一鍵部署（推送並同步到 GitHub Pages）

每次修改 `index.html` 後，執行以下指令：

```bash
git add index.html && git commit -m "update" && git push
```

推送完成後約 **30 秒 ~ 1 分鐘**，GitHub Pages 會自動更新。

## 常用指令

| 指令 | 說明 |
|------|------|
| `git status` | 查看哪些檔案有修改 |
| `git add .` | 加入所有變更 |
| `git commit -m "說明"` | 建立 commit |
| `git push` | 推送到 GitHub |
